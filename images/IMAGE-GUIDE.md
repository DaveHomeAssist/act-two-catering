# Image Replacement Guide — Act Two Catering

All placeholder images are in the `/images/` folder. To add your real photos,
just replace each file with your own photo **using the exact same filename**.

## Quick Steps

1. Take or select your photos
2. Rename them to match the filenames below
3. Drop them into the `/images/` folder (overwrite the placeholders)
4. Redeploy

## Recommended Photo Specs

| Type       | Size        | Orientation | Notes                          |
|------------|-------------|-------------|--------------------------------|
| Hero       | 1920×800 px | Landscape   | Dark/moody works best          |
| Menu Items | 600×400 px  | Landscape   | Overhead or 45° angle          |
| About      | 800×600 px  | Landscape   | Candid, warm lighting          |

JPG or PNG both work. Keep files under 500KB each for fast loading (compress at tinypng.com).

## Hero Images (background banners on each page)

These appear as large background images behind the page titles.
Dark or moody photography works best — text overlays on top.

| Filename                  | Used On        | What to Shoot                         |
|---------------------------|----------------|---------------------------------------|
| `hero-home.jpg`           | Home page      | Signature croquettes, beautifully plated |
| `hero-menu.jpg`           | Menu page      | Overhead shot of several dishes spread out |
| `hero-services.jpg`       | Services pages | Elegant event table setting           |
| `hero-events.jpg`         | Events pages   | Candid shot from a catered event      |
| `hero-reviews.jpg`        | Reviews page   | Happy guests at a table               |
| `hero-pricing.jpg`        | Pricing page   | Close-up of a plated dish             |
| `hero-about.jpg`          | About page     | Family/team cooking or at an event    |
| `hero-contact.jpg`        | Contact page   | Warm, inviting table setup            |
| `hero-areas.jpg`          | Service Areas  | Philly skyline, bridge, or local venue |

**Tip:** You can reuse the same photo for multiple hero slots. Just copy the file
with different names. 3–4 strong hero photos can cover all 9 slots.

## If You Don't Have a Photo for Every Slot

That's fine! The site has built-in fallbacks:
- **Hero images:** Falls back to a clean gradient (no image needed)
- **Menu items:** Falls back to the dish emoji icon
- **About photos:** Falls back to a labeled placeholder

You can delete any placeholder .jpg you don't want to replace yet and
the fallback will show automatically.
